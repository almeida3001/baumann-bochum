# KM-Bau GmbH

Cinematic one-page website for **KM-Bau GmbH** — family construction company in Gelsenkirchen since 2000.

Static site (HTML + CSS + vanilla JS, GSAP + ScrollTrigger + Lenis). Deployed on Vercel.

- Language: German (formal)
- Palette: black (#0C0C0D) + red (#E01E26) — client's identity
- No marquee/ticker strip (not wanted for construction sites)
- Display type: Poppins · Body: Inter
- Hero + final CTA images generated with Higgsfield; project gallery uses the client's real photos
